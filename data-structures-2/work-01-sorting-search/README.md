# Work 01 - Sorting and Searching Algorithms

## 📚 Course
Data Structures 2 - Computer Science (PUC Goiás)

---

## 📌 Description

This project implements and analyzes classical sorting and searching algorithms, focusing on performance and complexity behavior.

---

## ⚙️ Implemented Algorithms

### Sorting Algorithms:
- Bubble Sort
- Selection Sort
- Merge Sort

### Searching Algorithms:
- Sequential Search
- Binary Search

---

## 📊 Experiments

Tests were performed using:
- Input sizes: 100, 1000, 10000 elements
- Data types: ascending, descending, and random
- Metrics: execution time and number of comparisons

---

## 🧠 Main Objective

To compare the practical performance of different algorithms and relate the results to their theoretical time complexities.
