# 🧩 Object-Oriented Programming (CMP2108)

Repository containing exercises, assignments, and projects developed during the **Object-Oriented Programming** course at PUC Goiás.

---

## 📚 Course Information

* **Course Code:** CMP2108
* **Workload:** 60 hours
* **Credits:** 4
* **Language:** Java

---

## 🎯 Objectives

The course focuses on developing and implementing systems using the object-oriented paradigm in Java.

By the end of the course, students are able to:

* Develop applications using primitive and composite data types
* Apply object-oriented programming concepts in Java
* Represent and manipulate objects effectively
* Implement algorithms using OOP principles
* Work with files and string manipulation
* Document systems using UML

---

## 🧠 Topics Covered

* Object-Oriented Concepts:

  * Abstraction
  * Classes and Objects
  * Encapsulation
  * Inheritance
  * Polymorphism
  * Interfaces

* UML Class Diagrams:

  * Attributes, methods, visibility
  * Relationships (association, aggregation, composition, generalization)

* Java Programming:

  * Packages and scope
  * Exception handling
  * File manipulation
  * Collections

* Advanced Concepts:

  * Method overloading
  * Constructors and garbage collection
  * Abstract classes and interfaces
  * Cohesion and coupling
  * SOLID principles

* GUI Development:

  * Java Swing / JavaFX (optional)
