# 📊 Fundamentals of Computing IV (CMP1058)

Repository containing implementations, assignments, and projects developed during the **Fundamentals of Computing IV** course at PUC Goiás.

---

## 📚 About

This repository is a structured collection of academic work focused on **numerical methods** and **computational mathematics**, applied to solving complex problems through approximation techniques.

The course emphasizes both **mathematical understanding** and **computational implementation** using tools such as Python, MATLAB, or Octave.

---

## 🧠 Purpose

* Apply numerical methods to solve mathematical problems
* Analyze precision, efficiency, and computational limitations
* Implement algorithms for approximation and modeling
* Strengthen problem-solving using computational techniques
* Bridge mathematical theory and practical implementation

---

## 🎯 Objectives

By the end of the course, students are able to:

* Understand and analyze different types of numerical errors
* Solve nonlinear equations and systems
* Apply interpolation and curve fitting techniques
* Implement numerical methods for differentiation and integration
* Solve differential equations numerically
* Apply optimization methods
* Use computational tools to test and validate algorithms

---

## 📂 Topics Covered

* Error Analysis:

  * Absolute and relative error
  * Truncation and rounding errors
  * Error propagation

* Nonlinear Equations:

  * Bisection Method
  * False Position
  * Fixed Point Iteration
  * Newton-Raphson
  * Secant Method

* Linear and Nonlinear Systems:

  * Jacobi Method
  * Gauss-Seidel
  * Newton-based methods

* Interpolation and Approximation:

  * Newton Polynomials
  * Lagrange Interpolation

* Curve Fitting:

  * Linear Regression
  * Least Squares Method

* Numerical Differentiation:

  * Finite differences
  * Richardson extrapolation

* Numerical Integration:

  * Trapezoidal Rule
  * Simpson’s Rule
  * Gaussian Quadrature

* Optimization:

  * One-dimensional and multi-dimensional optimization

* Differential Equations:

  * Euler Method
  * Runge-Kutta Methods
  * Finite differences
