# 🌳 Data Structures II (CMP2109)

Repository containing implementations, assignments, and projects developed during the **Data Structures II** course at PUC Goiás.

---

## 📚 About

This repository is a structured collection of academic work focused on **non-linear data structures** and **algorithm analysis**, implemented in **C**.

The course emphasizes practical development, performance analysis, and deep understanding of how data structures operate internally.

---

## 🧠 Purpose

* Develop and implement algorithms in C
* Analyze time complexity and performance
* Understand and apply non-linear data structures
* Strengthen problem-solving and algorithmic thinking
* Build a solid foundation in data structures

---

## 🎯 Objectives

By the end of the course, students are able to:

* Estimate algorithm execution time
* Implement and analyze sorting and searching algorithms
* Develop hash tables with appropriate hash functions
* Implement and apply tree-based structures
* Work with binary trees and balanced trees
* Understand and implement B-Trees (secondary memory structures)
* Apply recursion and pointer manipulation effectively

---

## 📂 Topics Covered

* Review of pointers and recursion

* Time complexity analysis

* Searching algorithms:

  * Sequential Search
  * Binary Search

* Sorting algorithms:

  * Bubble Sort
  * Selection Sort
  * Insertion Sort
  * Merge Sort
  * Quick Sort

* Linear data structures:

  * Linked Lists
  * Doubly Linked Lists
  * Stack (LIFO)
  * Queue (FIFO)
  * Deque

* Non-linear data structures:

  * Trees
  * Binary Trees
  * Balanced Trees
  * B-Trees

* Hash Tables

## 🧪 Main Assignments

### 🔹 Assignment 1 – Sorting and Searching Algorithms

Implementation and experimental analysis of classic algorithms:

* Sequential Search
* Binary Search
* Bubble Sort
* Selection Sort
* Insertion Sort

📊 Includes:

* Performance comparison
* Number of comparisons and swaps
* Execution time analysis under different scenarios

---

### 🔹 Assignment 2 – Linear Data Structures System

Development of a complete system using:

* Linked Lists
* Stack or Queue
* Sorting and searching algorithms

💡 Features:

* Insert, remove, search, and display data
* Dynamic memory management
* Modular code organization (.h / .c)
* Interactive terminal menu

---

## ⚙️ Methodology

The course follows a **hands-on approach**, focusing on:

* Manual implementation (no external libraries)
* Step-by-step coding (live coding)
* Experimental testing with different data sizes
* Comparative analysis of algorithms
* Collaborative discussion and code review
