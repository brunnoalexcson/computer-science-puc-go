# 🕸️ Graph Algorithms (CMP2119)

Repository containing implementations, assignments, and projects developed during the **Graph Algorithms** course at PUC Goiás.

---

## 📚 About

This repository is a structured collection of academic work focused on **graph theory** and **graph algorithms**, covering both theoretical concepts and practical implementations.

The course explores how graphs can model real-world problems and how efficient algorithms can be applied to solve them.

---

## 🧠 Purpose

* Understand fundamental graph theory concepts
* Implement graph algorithms from scratch
* Analyze algorithm efficiency and complexity
* Solve computational problems using graph models
* Build a solid foundation for advanced topics (networks, optimization, AI)

---

## 🎯 Objectives

By the end of the course, students are able to:

* Understand core graph concepts (vertices, edges, degree)
* Represent graphs using different structures
* Implement and analyze classical graph algorithms
* Solve real-world problems using graph-based approaches
* Evaluate performance and complexity of algorithms

---

## 📂 Topics Covered

* Graph Fundamentals:

  * Vertices, edges, degree
  * Directed and undirected graphs
  * Weighted and unweighted graphs
  * Simple graphs and multigraphs

* Graph Representation:

  * Adjacency Matrix
  * Adjacency List
  * Incidence Matrix

* Graph Properties:

  * Connectivity
  * Planarity
  * Isomorphism
  * Subgraphs and cliques
  * Eulerian and Hamiltonian graphs
  * Bipartite graphs
  * Graph coloring

* Graph Traversal:

  * Depth-First Search (DFS)
  * Breadth-First Search (BFS)

* Advanced Algorithms:

  * Shortest Paths
  * Maximum Flow
  * Matching Algorithms
